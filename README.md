# Keys

> e -> create random grid
> 
> r -> reset
>
> p -> print
> 
> space -> pause/resume
> 
> up key -> increase cell size
> 
> down key -> decrease cell size

# Build

> cmake -B build
>
> cmake --build build
>
> ./build/GameOfLife
